# Product Overview

TicketFlow is a modern, production-ready ticket management system designed for customer support and issue tracking.

## Core Features

- Dashboard with real-time statistics and ticket metrics
- Full ticket lifecycle management (create, edit, assign, archive, delete)
- Category-based organization with priority levels (low, medium, high)
- User management with role-based access control (admin, agent, customer)
- Comment system for ticket discussions
- Activity logging for audit trails
- Responsive design for mobile and desktop

## User Roles

- **Admin**: Full system access, user management, ticket assignment
- **Agent**: Can view and manage assigned tickets
- **Customer**: Can create and view their own tickets

## Deployment Options

- One-click Vercel deployment with serverless functions
- Local development with Laravel backend + Vue frontend
- Includes sample data (144 tickets, 31 users) for testing
